# 💡 Idea Canvas — Lean Canvas (4 ช่องแรก)

> ทำช่วงท้ายวัน เปลี่ยน "ของที่รันได้วันนี้" → "ไอเดียนวัตกรรมที่ต่อยอด Day 2 ได้"
> เริ่มจาก **ปัญหา** ไม่ใช่ "เรามี sensor อะไร" · ทำ 4 ช่องนี้ให้คมก่อน ที่เหลือไว้ทีหลัง

- ชื่อ Prototype (ชั่วคราว): ระบบป้อมปืนปัญญาประดิษฐ์สำหรับตรวจจับและขับไล่สัตว์รบกวนในสวนทุเรียน

## 1️⃣ Problem — ปัญหาคืออะไร

- การบุกรุกพื้นที่ของสัตว์รบกวน โดยเฉพาะฝูงลิงที่เข้าทำลายผลผลิตทางการเกษตรมูลค่าสูง เช่น ทุเรียน ในระยะเวลาอันสั้น ได้ก่อให้เกิดความสูญเสียทางเศรษฐกิจต่อเกษตรกรอย่างมีนัยสำคัญ
- มาตรการผลักดันแบบดั้งเดิมขาดประสิทธิภาพในระยะยาว เนื่องจากสัตว์สามารถเรียนรู้ ปรับพฤติกรรม และเกิดความเคยชินได้
- เกษตรกรขาดแคลนกำลังคนในการเฝ้าระวังตลอด 24 ชั่วโมง และมีข้อจำกัดทางกฎหมายคุ้มครองสัตว์ป่าที่ห้ามใช้มาตรการขั้นเด็ดขาดอันก่อให้เกิดอันตรายถึงชีวิตสัตว์

**ตอนนี้คนแก้ปัญหานี้ยังไง (ของเดิม)?** ใช้แรงงานมนุษย์ในการลาดตระเวนเฝ้าระวัง, การขึงตาข่าย, การจุดประทัดขับไล่ด้วยตนเอง หรือการใช้สัตว์เลี้ยงเฝ้าสวน ซึ่งมีต้นทุนสูงและให้ผลลัพธ์เพียงระยะสั้น

## 2️⃣ Customer — ใครคือผู้ใช้จริง

เกษตรกรผู้เพาะปลูกทุเรียนและผลไม้เศรษฐกิจเชิงพาณิชย์ ที่มีพื้นที่เพาะปลูกรอยต่อติดแนวป่าหรือภูเขา และประสบปัญหาความสูญเสียจากการบุกรุกของสัตว์รบกวนอย่างต่อเนื่อง

## 3️⃣ Value (USP) — คุณค่าเด่นในประโยคเดียว

Smart Durian Guardian ช่วยเกษตรกรผู้เพาะปลูกทุเรียนขับไล่สัตว์รบกวนออกจากพื้นที่เพาะปลูกได้โดยอัตโนมัติตลอด 24 ชั่วโมง ต่างจากการใช้แรงงานคนหรือมาตรการดั้งเดิมเพราะเป็นระบบป้อมปืนปัญญาประดิษฐ์เชิงรุกที่สามารถหมุนติดตามเป้าหมายและใช้มาตรการฉีดน้ำแรงดันสูงร่วมกับคลื่นเสียงผลักดันเฉพาะจุดได้อย่างแม่นยำ โดยปราศจากการใช้ความรุนแรงและไม่รบกวนมนุษย์

## 4️⃣ Solution — แก้ด้วยอะไร

Input ที่ใช้: โมดูลรับภาพประมวลผล (กล้อง logitech)
Class ที่ AI แยก: ลิง (Monkey), มนุษย์ (Person) และ สภาพแวดล้อมทั่วไป (Background)
Output ตอบสนอง: เซอร์โวมอเตอร์ 2 แกน (Pan-Tilt) สำหรับหมุนติดตามเป้าหมาย, ระบบควบคุมวาล์วปั๊มน้ำ (Water Pump/Relay) และ อุปกรณ์กำเนิดเสียงความถี่สูง (Buzzer/Speaker)

## 🎯 MVP — ฟังก์ชันเดียวที่ต้องสาธิตให้ได้

```
Input (ภาพจากกล้อง) → AI Model → บอร์ดประมวลผล → Prediction (สกัดพิกัดเป้าหมาย) → Output (Servo หมุนป้อมปืนติดตามเป้าหมาย + จำลองการเปิดวาล์วน้ำและเสียงไซเรน)
```

เมื่อระบบตรวจพบภาพลิง อัลกอริทึมจะสกัดพิกัดเชิงพื้นที่เพื่อสั่งการให้เซอร์โวมอเตอร์หมุนป้อมปืนติดตามเป้าหมาย พร้อมส่งสัญญาณกระตุ้นการทำงานของกลไกจำลองการฉีดน้ำและเสียงไซเรนขับไล่ทันที ในทางกลับกัน หากระบบตรวจพบมนุษย์ ป้อมปืนจะรักษาสถานะหยุดนิ่งและไม่มีการตอบสนอง เพื่อยืนยันความแม่นยำในการจำแนกประเภทเป้าหมายและการประมวลผลของระบบ

> คาถา: **Build less, prove more** — ฟังก์ชันเดียวที่ทำงานจริง + เล่าได้ ดีกว่าหลายอย่างที่ทำไม่สุด
